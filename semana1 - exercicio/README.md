# Exercício Semana 1 – C#

Este projeto contém um exercício simples em C# para praticar entrada de dados, condicionais e operações matemáticas básicas no console.

## Descrição
O programa está dividido em duas partes principais:

### Parte 1: Análise de Número Inteiro
- Solicita ao usuário um número inteiro.
- Informa se o número é positivo, negativo ou zero.
- Informa se o número é par ou ímpar.

### Parte 2: Calculadora Simples
- Solicita dois números reais (aceita ponto como separador decimal).
- Pergunta ao usuário qual operação deseja realizar:
  - `1` para somar
  - `2` para subtrair
  - `3` para sair
- Exibe o resultado da operação escolhida ou uma mensagem de opção inválida.

## Como Executar
1. Certifique-se de ter o [.NET 8.0 SDK](https://dotnet.microsoft.com/download) instalado.
2. Abra o terminal na pasta do projeto.
3. Execute o comando:
   ```sh
   dotnet run
   ```
4. Siga as instruções exibidas no console.

## Estrutura do Projeto
- `Program.cs`: Código-fonte principal do exercício.
- `semana1 - exercicio.csproj`: Arquivo de configuração do projeto.
- `README.md`: Este arquivo de instruções.

## Observações
- O programa trata entradas inválidas sem lançar exceções.
- Os números reais devem ser digitados usando ponto como separador decimal (ex: 3.14).
